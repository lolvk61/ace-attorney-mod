# Ace Attorney Mod

Fabric-мод для Minecraft 1.21.11 в духе Ace Attorney: судебные заседания, улики и выкрики «OBJECTION!».

## Возможности

### Выкрики (работают в мультиплеере)
Нажми клавишу — все игроки в радиусе 64 блоков увидят плашку на весь экран и услышат звук:

| Клавиша | Выкрик |
|---------|--------|
| `O` | OBJECTION! (Протестую!) |
| `H` | HOLD IT! (Минуточку!) |
| `J` | TAKE THAT! (Вот!) |

Клавиши можно переназначить в настройках управления (категория «Ace Attorney»).

### Судебные заседания — команда `/court`
- `/court start` — начать заседание (ты становишься судьёй)
- `/court role <игрок> <judge|defense|prosecution|witness|defendant>` — назначить роль (только судья)
- `/court roles` — список участников
- `/court evidence add <"название"> <описание>` — приобщить предмет в руке к делу
- `/court evidence list` — материалы дела (наведи курсор на улику для описания)
- `/court evidence remove <номер>` — изъять улику (только судья)
- `/court present <номер>` — предъявить улику («TAKE THAT!» на весь экран)
- `/court verdict guilty` / `/court verdict notguilty` — вердикт с титрами (только судья)
- `/court end` — завершить заседание

### Диалоговые окна
- `/aa say <текст>` — реплика в текстбоксе в стиле AA (печатающийся текст со звуком)
  для всех игроков в радиусе 32 блоков. Окна выстраиваются в очередь.

### Перекрёстный допрос (в рамках `/court`)
- `/court testimony add <текст>` — свидетель записывает показание (по одному)
- `/court testimony play` — титр «ПОКАЗАНИЯ СВИДЕТЕЛЯ» и проигрывание всех показаний по очереди (зелёным)
- `/court testimony list` — список показаний в чате
- `/court testimony clear` — очистить показания (только судья)
- `/court press <№>` — «HOLD IT!» + требование подробностей по показанию
- `/court object <№ показания> [№ улики]` — «OBJECTION!» + заявление о противоречии;
  решение о том, засчитан ли протест, принимает судья

### Предметы (вкладка «Ace Attorney» в креативе)
- **Значок адвоката** — ПКМ: гордо показать значок окружающим
- **Магатама** — ПКМ: почувствовать, скрывает ли кто-то правду
- **Судейский молоток** — ПКМ: стук молотка и «Порядок в зале суда!»

## Сборка

```
./gradlew build
```

Готовый jar: `build/libs/aceattorney-<версия>.jar`. Положи его в папку `mods`
вместе с [Fabric API](https://modrinth.com/mod/fabric-api). Мод нужен и на сервере, и на клиентах.

## Свои звуки

Сейчас выкрики озвучены плейсхолдерами из ванильных звуков (см.
`src/main/resources/assets/aceattorney/sounds.json`). Чтобы поставить свои:

1. Положи файлы `objection.ogg`, `hold_it.ogg`, `take_that.ogg` в
   `src/main/resources/assets/aceattorney/sounds/`
2. В `sounds.json` замени запись вида
   `{ "name": "minecraft:entity.ravager.roar", "type": "event" }`
   на `{ "name": "aceattorney:objection" }`

⚠️ Оригинальные звуки и спрайты Capcom защищены авторским правом — не включай их
в публичные сборки мода.
